## Summary

When using React, you can still use the library **as you would in a normal JavaScript project**, but to make it easier to use, we provide a few helpers.

The mainly difference is that internally it uses `useMemo` and `useCallback` to memoize the results and the search function and also it provides a **easier** `API` to use.

The main new concepts are `useFuzzy` and `Highlight`.

## Basic Example

With only a **few lines of code** you can create a fuzzy search function that will return the results based on the query and **highlight the matching characters**.

```tsx title="App.tsx"
import { useFuzzy, Highlight } from '@polgubau/fuzzy/react' // note we are now importing from /react

const filteredList = useFuzzy({list: ["volvo", "seat", "mercedes", "audi", "bmw"], query: "volv"})

filteredList.map(({ item, matches: [range] }) => (
  <div key={item.key}>
    <Highlight text={item} ranges={range} /> 
  </div>
))
```

## API
### `useFuzzy`

This is the main hook, it provides a performance optimized way to search through a list of items.

The available props are the same as the [`fuzzy` function](/get-started), but with a few extra options:

1. list - The list of items to search through. <small>Required</small>.
2. query - The query to search for. <small>Required</small>.

<Callout> Note how now you are using one step only, you don't need to create the `fuzzy` function and then call it with the query. You can just use `useFuzzy` and it will do it for you internally.</Callout>

### All `useFuzzy` Options

```tsx title="App.tsx"
const filteredList = useFuzzy({
  list, // required, the list of items to search through
  query, // required, the query to search for (if empty, will return all items)
  key: "name", // optional, the key if items are objects 
  getKey: (item) => [item.name, item.description], // optional, the key if items are objects (same as key but can be a function or multiple keys)
  mapResult: ({ item, score, matches: [nameRange] }) => ({ item, nameRange })  // arbitrary mapping function, takes `Result<T>` as input
  debug: true, // optional, if true will log the results to the console
  limit: 10, // optional, the maximum number of results to return  
  maxScore: 0.5, // optional, the maximum score to return 
})
```
### Highlight
This is a helper component that will highlight the matching characters in the text.

You could perfectly recreate this component yourself, but it's provided for convenience.

```tsx  title="App.tsx"
// @noErrors 
import { useFuzzy, Highlight } from '@polgubau/fuzzy/react'

const App = () => {
	const list = ["volvo", "seat", "mercedes", "audi", "bmw"];
	const query = "volv";
	const { results } = useFuzzy({ list, query });

	return (
		<ul>
			{results.map(({ item, matches }) => (
				<li key={item}>
						<Highlight text={item} ranges={matches[0]} />
//           ^^^^^^^^^
				</li>
			))}
		</ul>
	);
};
```
