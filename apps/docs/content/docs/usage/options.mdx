## Fuzzy list options

To customize the fuzzy, it accepts an object with the following options:
```ts   
const fuzzySearch = fuzzy(list, {
  key: "name", // This is the key to search for in the item. 
  getText: (item) => item.name, // This is a function that returns the string to search. 
  debug: true, // Add any additional options here
  limit: 10, // The maximum number of results to return
  maxScore: 2.5, // The maximum score for the results
});
```

In case the data sent are objects, you can use the `key` and `getText` options to specify which property to search for.
